Cloud Computing: the delivary of in-demand IT resources over the internet with pay-as-you-go pricing.
- User are able to rent computing power, storage, server, databases, AI, ML tools, etc
- User also don't need to purchase hardware resources for it
- User able to access resources within seconds through internet connectivity

Cloud deployment types
- On-cloud: all data are migrated to the cloud , will tuilize the most of cloud infrastructure
- On-premise:utilise the least of cloud infrastructure and used only to try increasing resource utilization
- Hybrid: using both on-cloud and on-premises, can also refer to multi-cloud reployment

AWS Global Infrastructure means that resources will all around. If a disaster happens, there is another one ready to pick up the slack. This makes the resources fault tolerence.
